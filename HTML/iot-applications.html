Activity 1 : IOT APPLICATION

IoT applicationss are software programmes that run on IoT devices or connect different IoT devices, to fullfil multiple or specific goals. Some primary goals of IoT Applications are to -

  • Displays the data collected from sensors and other devices on its network/database
  • Statistically organizing the data
  • Using the collected data to execute a process
